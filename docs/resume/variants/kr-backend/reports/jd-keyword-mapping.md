## 바리에이션별 반영 전략

### v1. JD 매칭형

- 채용담당자/현업 리더가 빠르게 체크하는 키워드와 성과지표를 상단에 배치
- 문장 길이를 줄이고 프로젝트당 "문제-조치-성과" 3줄 구조 유지

### v2. 문제해결형

- 면접 질문으로 이어지기 쉬운 문제 해결 흐름(상황-판단-실행-결과) 강화
- 운영 사고/복구 대응 역량을 사례 단위로 분리

### v3. 아키텍처형

- 기술 면접 대응을 위해 설계 원칙, 트레이드오프, 운영 패턴을 전면 배치
- 멱등, Retry, DLQ, Adapter 등 키워드를 실제 경력 문장과 연결
