## 작성 가이드

새 일기를 작성할 때:

1. `docs/diary/` 폴더에 `YYYY-MM-DD.md` 형식으로 파일 생성
2. 아래 템플릿을 복사해서 사용

```markdown
# YYYY-MM-DD

## Summary
오늘의 한 줄 요약

## What I Learned
- 배운 내용

## What I Did
- 진행한 작업

## Thoughts
생각이나 아이디어

## TODO
- [ ] 내일 할 일
```
